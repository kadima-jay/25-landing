因果推断：
从 反事实 到现代方法

以电商数据分析师视角，构建从 AB 实验到双重机器学习的方法体系。不堆公式，讲清每个方法要回答的根本问题。

01 / 根本矛盾

你永远只能看到硬币的一面

因果推断只有一个核心问题：给某个用户施加某个策略，结果改变了多少？

个体因果效应 τᵢ = Yᵢ(1) − Yᵢ(0)，但同一个用户你永远只能观测到一种状态——收到策略，或没收到。观测不到的那个，就是反事实（counterfactual）。

就像你没法同时知道你读研和不读研两条人生线上各自会赚多少钱。走了其中一条，另一条就永远消失了。

解决思路是退一步：不追求个体因果效应，转而估计平均因果效应（ATE）。

ATE = E[Y(1)] − E[Y(0)]

AB 实验如何填上这个窟窿

随机分流 → 两组在所有特征上分布相同 → 对照组的平均结果就是实验组「如果没被干预」的无偏替身。随机化是唯一不需要任何假设就能识别因果效应的办法。

02 / 当 AB 做不了时

三条路绕开混杂因子

核心障碍：混杂因子（confounder）——同时影响「是否被处理」和「结果」的变量。没有随机化，处理组和对照组就不再可比较。

路径核心假设典型方法适用场景
条件独立观测并控制所有混杂因子回归、PSM、匹配有丰富用户特征数据
双重差分未观测混杂因子趋势不变DiD有前后两期面板数据
工具变量存在只通过处理影响结果的随机冲击2SLS有天然随机外部变量

路径一：条件独立假设

假设控完全部可观测协变量 X 后，处理分配就像随机一样：(Y(0), Y(1)) ⟂ T ∣ X

方法一

回归

把 X 扔进方程，β₁ 就是因果效应。暗坑：强制假设 Y 和 X 的线性关系。

方法二

匹配 Matching

给实验组每个人在对照组找「长得像的」。暗坑：高维空间里「长得像」越来越难——维数诅咒。

方法三

倾向得分 PSM

把高维 X 压缩成一维概率 e(X)=P(T=1|X)，然后匹配/加权。优势：绕开维数诅咒。

路径二：双重差分 DiD

不做「我控全了」的假设，改做更弱的假设：如果没有策略，处理组和对照组会以相同趋势变化（平行趋势假设）。

DiD = (处理后 − 处理前) − (对照后 − 对照前)
第一层差分去组间固有差异，第二层差分去时间趋势。两次减法绕开不可观测混杂因子。

可按策略前趋势是否平行来部分检验假设——这是 DiD 相比条件独立路径的核心优势。

路径三：工具变量 IV

找一个只通过策略来影响结果、但本身与混杂因子无关的变量。三条条件缺一不可：

条件含义电商案例：配送速度→复购
相关性IV 与处理变量相关暴雨 → 配送延迟 ✓
外生性IV 与混杂因子无关暴雨与用户消费力无关 ✓
排他性IV 只通过处理变量影响 Y暴雨只通过配送延迟影响复购 ✓

IV 在电商里少见的原因：相关性和外生性天然打架。能影响用户行为的变量，往往也跟用户特征相关。

优先级排序：

  1. 能做 AB → 做 AB

    随机化解决一切，无需任何假设。

  2. AB 做不了、有面板数据 → DiD

    平行趋势假设比「我控全了所有变量」更可信。

  3. 面板数据也没有 → PSM/回归

    在可观测变量假设下做近似推断。

  4. 有干净外部冲击 → IV

    最后一张牌。没有干净的 IV 就认。

03 / 三条铁律

数据的局限比方法的局限更大

铁律一

避免过拟合

因果推断的过拟合比预测模型更致命——预测过拟合只是 AUC 掉几个点，因果过拟合直接导致错误决策。必须 holdout 验证，仅靠 p 值筛选 = p-hacking。

铁律二

检查异质性

ATE 掩盖真实故事。平均正确 ≠ 对每个人都正确。异质性是信号不是噪音——同一个策略对不同人群效果不同。新用户 +20、老用户 -3，全量推就是一边赚一边亏。

铁律三

稳健性检验

换方法（回归 vs PSM vs DiD）、换窗口（切掉首尾 5%）、安慰剂检验（把策略时间前移，效应消失了吗？）。用这三条回答老板的「你确定？」。

04 / 现代因果推断

机器学习怎么入局

传统方法（PSM、DiD、IV）来自计量经济学，有 50 年历史。近十年，机器学习把这些方法升级了一次。核心变化：传统方法需要你手动指定 Y 和 X 的关系形式，现代方法让数据自己学。

双重机器学习 DML

直觉：把机器学习当「混杂因子吸尘器」，吸干净后剩下的就是干净的因果信号。

  1. 拿所有 X 用 ML 预测 Y，取残差

    得到 X 解释不了的 Y 波动

  2. 拿所有 X 用 ML 预测 T，取残差

    得到 X 解释不了的 T 波动

  3. 残差 Y 对残差 T 回归

    系数就是因果效应，有严格置信区间

优势：不需要手动写非线性项和交互项；有严谨统计推断；交叉拟合控制了过拟合风险。

因果森林 Causal Forest

把决策树的切分逻辑从「哪个特征最能区分 Y 差异」改成「哪个特征最能区分因果效应差异」。多棵树取平均 → 因果森林 → 每个用户得到自己的个体因果效应（CATE）。

异质性分析三层

层级方法关键点
第一层预设维度分组 + BH 校正你现在能做的，但记得多重比较校正
第二层因果森林自动发现数据驱动，不拍脑袋选分组维度
第三层CATE 指导后续定向实验需要百万级样本，把上一轮学到的用于下一轮

05 / 深层认知

三条暗线

I

因果推断不是在找因果，是在找随机化的替代品

AB 实验靠物理随机消除反事实，DiD 靠时间差分逼近，PSM 靠条件独立假设——所有方法都是试图在「没有随机」的数据里重建「像是随机」的条件。判断一个方法是否可靠，就看它离随机化有多近。

II

平均正确是决策毒药

ATE 让你觉得能放心做全量决策，但它掩盖了谁受益、谁受害的真相。因果推断的真正价值不在估计一个数字，而在发现「对谁有效、对谁无效」。异质性不是锦上添花——它是因果分析的真正产出。

III

方法的选择权在数据，不在分析师

不是你选 PSM 还是 DiD，是你的数据结构决定你能走哪条路。有 AB 就走 AB，有面板就走 DiD，都缺就只能在不完美的假设下做近似推断。真正专业的人知道什么时候认——当数据不支持因果识别时，诚实比精巧更重要。

06 / 速查

关键概念

ATE平均因果效应，所有人平均
CATE条件平均因果效应，不同人群各自效应
反事实观测不到的那个状态
混杂因子同时影响 T 和 Y 的变量
条件独立控完 X 后，处理和随机一样
平行趋势没策略的话，两组趋势相同
异质性同一策略不同人效果不同——是信号，非噪音
DMLML 当混杂吸尘器，残差对残差回归
因果森林找因果效应差异最大切分点的随机森林
安慰剂检验把策略时间前移，效应应消失
